기획전 상품 등록 템플릿 자동화 제작

기존 : 퍼블리셔가 HTML 이벤트 페이지를 제작해서 기획전을 생성
퍼블리셔의 공수가 많이 필요했으며 (이벤트담당 퍼블 : 3명) MD가 긴급수정시에 어려움이 있었음

분석 : 이벤트 기획전은 크게 1.이미지 / 2.일반링크(프로모션) / 3.롤링상품(상품데이터) 이 주력이었으며
MD가 직접 기획전을 만들고 사용할수 있게 만들수 있을것 같다고 판단하여 2달간 운영업무와 병행하며 진행하겠다고 승인 받음

설계 : UI&UX기반을 고려한 설계서를 직접 작성함

진행 : 백엔드개발1명(지인 외주)과 프론트개발+퍼블리싱1명(나)으로
둘이서 8주간 기획전 자동화 페이지를 제작

테스트 : MD와 비개발자(디자이너,지원부서등등)에게 템플릿 자동화 교육 및 체험 시작

기능추가 : 유저들의 체험후기를 통해 추가 편의 기능 및 특별 기능(동영상,패럴랙스크롤)등등 추가

오픈 : 오픈 후 이벤트 페이지 제작 공수 7~80% 감소

하프클럽

  • 작업 범위 : PC , Mobile
  • Project URL PC : https://www.halfclub.com
  • Project URL MO : https://nm.halfclub.com

리뉴얼 및 운영

소스 리팩토링 : FO가 angular1 로 제작이 되어있는데, 당시 IE 브라우저의 느린증상이 심각한 수준이었음
클라이언트 스크립트단의 문제를 발견하여, 성능을 개선시킴 (스크롤이벤트시마다 eventlistener 계속발생)
무분별한 timeout 제거, 필요없는 3중 반복문을 제거후 적용
많은수의 JS 라이브러리를 제거 후 CSS 애니메이션과 pure JS 로 변환

운영 템플릿 제작 : Parallax , mini game , effect의 템플릿 사용가이드와 함께 제작

동영상 플랫폼 제작 : Video.js로 HTTP 라이브 스트리밍 (HLS) 사내 플랫폼 제작

UI Framework 도입 : 기존엔 pure css로 중복 되는 CSS와 오류 및 작업 효율이 상당히 떨어져 있었음
SCSS 와 ES6 자동화 빌드 시스템 환경 구축과 이미지 스프라이트를 자동화로 변환 시킴

FO 운영 및 팀원 리딩 : angular1 프론트 운영 작업과 신입 퍼블(4명)을 교육 및 리딩
